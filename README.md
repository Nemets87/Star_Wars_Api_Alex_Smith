# 🌟 Star Wars: Друзья Дарта Вейдера

Автотест, который находит всех персонажей, когда-либо снимавшихся вместе с Дартом Вейдером, и сохраняет их имена в текстовый файл без дубликатов.

Проект создан в учебных целях, но реализован по всем канонам профессиональной разработки:
- Чистый Python-код с комментариями и аннотациями типов.
- Логирование каждого шага (в файл и консоль).
- Allure-отчёт с подробными шагами и вложениями.
- Готовый CI/CD (GitHub Actions) для автоматического запуска и публикации отчёта на GitHub Pages.

## Как это работает
Скрипт обращается к [SWAPI — Star Wars API](https://swapi.dev/):
1. Получает данные Дарта Вейдера (`/people/4/`).
2. Проходит по всем фильмам, в которых он участвовал.
3. Собирает уникальные ссылки на персонажей этих фильмов.
4. Запрашивает имя каждого персонажа.
5. Сортирует имена и сохраняет их в файл `vader_friends.txt`.
6. Каждый HTTP-запрос проверяется на статус-код (должен быть 200).
7. Все действия записываются в лог-файл `vader_mission.log`.
8. Генерируется подробный Allure-отчёт.

## Требования
- Python 3.8 или новее
- Git
- (опционально) Allure CLI для локального просмотра отчёта

## Установка и запуск локально
```bash
# 1. Клонировать репозиторий
git clone https://github.com/твой-ник/имя-репозитория.git
cd имя-репозитория

# 2. Создать и активировать виртуальное окружение
python -m venv venv
source venv/bin/activate   # macOS/Linux
# или venv\Scripts\activate  # Windows

# 3. Установить зависимости
pip install -r requirements.txt

# 4. Запустить тест
pytest test_vader.py --alluredir=./allure-results -v

# 5. Посмотреть Allure-отчёт (если установлен Allure CLI)
allure serve ./allure-results
